# Api-Contato-Springboot
# 📒 Agenda de Contatos - Spring Boot API

Sistema de gerenciamento de contatos pessoais desenvolvido com Spring Boot.
Este projeto tem como objetivo oferecer uma API RESTful para operações CRUD em uma agenda de contatos, permitindo cadastrar, editar, listar, buscar e excluir contatos.

---

## 📌 Funcionalidades

- ✅ Cadastrar novo contato
- ✅ Listar todos os contatos
- ✅ Buscar contato por ID
- ✅ Buscar contatos por nome, e-mail ou telefone
- ✅ Atualizar dados de um contato
- ✅ Excluir contato
- ✅ Validações com Bean Validation
- ✅ Documentação da API com Swagger
- ✅ Mapeamento com DTOs e uso de ModelMapper
- ✅ Tratamento de exceções personalizado

---

## 🧪 Tecnologias Utilizadas

- Java 21
- Spring Boot
- Spring Web
- Spring Data JPA
- Bean Validation
- ModelMapper
- Lombok
- Banco de Dados H2 e PostgreSQL
